
# Coconut Simulator

Viva a experiência única de ser um coco em um ambiente 3D imersivo! Acumule pontos, explore cenários tropicais e dispute sua posição no ranking global de jogadores

# Sobre o jogo

Coconut Simulator é um jogo web desenvolvido com tecnologias modernas que oferece uma experiência única em primeira pessoa onde você assume o papel de um coco em um ambiente tropical dinâmico.

A aplicação é uma sátira onde simulamos a vida de um coco, e o modo realista exibe uma página completamente preta (pois coco não tem olho).
